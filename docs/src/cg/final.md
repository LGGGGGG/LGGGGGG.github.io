# 感想

盛名之下，其实难副。纵然 games101 有诸多好评，但是从框架的稳定性、一致性、易用性上看绝对不如我本科的设置(张宇师兄强！openGL 强！QT 强！MIT 强！)，我认为这门课是给那些学校开不出合格图形学课的学生学的。

但是不得不说这门课的 lecture 强于我本科图形学老师，年轻老师更近嘛。并且确实足够前沿，哥们补了很多当年没学到、没学好的知识。

还有一点是，这门课的作业其实更像是做给面试官看的，相当于一个公开数据集、一个 benchmark 而已，有一个公开证明对象，也许这就是当前求职环境的悲哀吧。

Anyway，算是做完了，我认为浙大某个本科生对这些东西的评论很准确：“做多了这些课程的 lab 以后，你就会发现这些都是玩具，做多了一两个以后就不会想再做了，反而回去追求真实项目。”
